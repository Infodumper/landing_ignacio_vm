# Web Personal de Promoción: Directivas de Diseño

## Principios Fundamentales
- **Estética Premium**: Inspiración en Apple/Linear. Fondos oscuros, tipografía sans-serif elegante (Outfit/Inter), y espaciado generoso.
- **Interactividad Viva**: Micro-animaciones en botones, efectos de cristal (blur), y transiciones suaves.
- **Enfoque en Conversión**: Call-to-actions claros para contacto y portafolio.

## Paleta de Colores
- **Fondo**: `#0a192f` (Deep Navy)
- **Texto Primario**: `#ccd6f6`
- **Acento (Links/Botones)**: `#64ffda` (Teal/Cyan)
- **Bordes**: `rgba(100, 255, 218, 0.1)`

## Componentes UI
- Utilizar `backdrop-filter: blur(8px)` para tarjetas.
- Bordes redondeados suaves (`12px` - `16px`).
- Sombras suaves (`drop-shadow`).
