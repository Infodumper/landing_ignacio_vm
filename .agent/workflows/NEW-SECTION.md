---
description: Crear una sección nueva en la página principal.
---

1. Definir el propósito de la sección (Hero, About, Portfolio, Contact).
2. Consultar `.agent/directives/UI-RULES.md` para asegurar consistencia visual.
3. Crear el componente en `src/components/` con su CSS modular.
4. Importar e integrar en `App.jsx`.
5. Ejecutar `npm run build` para validar estructuralmente.
