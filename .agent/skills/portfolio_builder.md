# Skill: Portfolio Builder

Este Skill capacita al Agente para mantener y expandir el porfolio de KAIZUNA.AI bajo los estándares más altos de la industria.

## Competencias Técnicas

- **Diseño de Interfaz**: Construcción de layouts de pantalla dividida (Split Screen), grillas de servicios de alta precisión y Hero Sections cinemáticos.
- **Implementación PHP**: Manejo de rutas limpias entre `index.php` y `metodologia.php` con persistencia de variables de diseño.
- **Micro-interacciones**: Dominio de efectos `hover`, `glow` (resplandor) y transiciones de tema (Dark/Light).
- **Asimilación de Marca**: Aplicación constante de los colores `@ff4d4d` (Coral), `#f59e0b` (Amber) y `#10b981` (Emerald).

## Reglas Críticas

1. **Mantener el "Boxy" Style**: Border-radius de 6px en botones y 1rem en tarjetas. Prohibidas las puntas redondas tipo "pill".
2. **Icons**: Siempre usar SVGs inline con colores temáticos del servicio correspondiente.
3. **Typography**: Mantener `Outfit` para impacto y `Inter` para legibilidad técnica. Letras siempre limpias y con espaciados calculados.
4. **PC Optimization**: El contenedor base debe centrarse en heredar el `80rem` (1280px) de diseño equilibrado.
