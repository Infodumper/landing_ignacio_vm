# Skill: Especialista Progresivo en Web Personal

## Descripción
Este skill otorga la capacidad de construir y evolucionar la web personal de promoción con un enfoque en:
1. **Promoción de Identidad**: Destacar los logros y habilidades del usuario.
2. **Optimización de Conversión**: Estrategias de captación de clientes a través de la web.
3. **Escalabilidad**: Estructura preparada para añadir blog, trabajos y servicios en el futuro.

## Capacidades
### 1. Construcción de Secciones
Capacidad de generar componentes visualmente impactantes siguiendo las `UI-RULES.md`.
### 2. Redacción de Copys
Escritura persuasiva para la sección Hero, Sobre mí, y Servicios.
### 3. SEO & Rendimiento
Optimización automática de etiquetas meta y velocidad de carga.

## Modo de Uso
Se debe llamar a este skill para:
- Crear una nueva sección.
- Editar el diseño del dashboard/home.
- Mejorar el branding.
