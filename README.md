# landing_ignacio_vm

**landing_ignacio_vm** es el repositorio que aloja la página web personal y el portafolio profesional de Ignacio Vizoso. El sitio está diseñado para mostrar servicios de consultoría, desarrollo de software e inteligencia artificial orientados a empresas, destacando casos de éxito y soluciones a medida.

## 🚀 Sobre el Proyecto

La landing page tiene como objetivo principal servir como carta de presentación profesional, explicando los tres pilares de enfoque:
1. **Datos:** Tableros y estructuración de la información para toma de decisiones.
2. **Automatización:** Optimización de flujos y reducción de tareas manuales.
3. **Agentes IA:** Implementación de asistentes inteligentes y procesos cognitivos.

Además, el sitio cuenta con secciones de:
- **Soluciones:** Sistemas empresariales, Business Intelligence e Inteligencia artificial.
- **Casos de Éxito:** Ejemplos de implementación real como CliP26, CRM de Fidelización, Dashboard Comercial y Consola Cognitiva.
- **Blog:** Espacio de artículos y novedades.
- **Contacto:** Para agendar reuniones y comenzar nuevos proyectos.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructuración semántica de las páginas.
- **CSS3:** Estilos personalizados (`styles/human.css`) sin frameworks invasivos, priorizando diseño limpio, modo oscuro/claro y glassmorphism.
- **Fuentes:** Inter, Crimson Pro, y Outfit.
- **Iconografía:** Font Awesome 6 y recursos gráficos personalizados.

## 📂 Estructura del Repositorio

- `index.html`: Página principal con el resumen de servicios, casos de éxito y llamado a la acción.
- `styles/`: Archivos CSS, incluyendo `human.css` con el sistema de diseño completo.
- `soluciones/`: Páginas de detalle de los servicios ofrecidos (Datos, Automatización, IA, etc.).
- `casos/`: Páginas de detalle de proyectos realizados.
- `blog/`: Sección de artículos y publicaciones.
- `public/`: Assets generales como imágenes e íconos utilizados en el sitio.
